Java起跑——HelloWorld

2021-05-15

作为Java初学者，接下来很长时间我将把握的学习历程以公众号和blog的形式进行记录，希望大家进行监督。由于是初学，所以有不足指出还望大家批评指正。
这段时间以来，捣鼓了好长时间终于把Java环境和eclipse配置和安装好了，虽然许多大佬建议从文本编辑器进行最初联系，但我还是打算直接用eclipse进行Java代码编写和学习，主要还是想早点接触eclipse，能够更快速的对接工作，再加上之前学习C语言时已经学习过直接文本编辑，所以这次学习Java则直接跳过了。
本次学习主要是敲下了任何一门语言的入门代码HelloWorld。
首先打开eclipse
在选项栏里File→New→Java Project就会出现下图

Project name里面填写本项目的名字，然后选定Location作为项目保存地方，这个是我专门建的一个Java学习文件夹，在Use an execution enviroment JRE选择你对应安装的jre版本，如果不知道的可以在终端控制台键入java -version进行查询，其他的暂时不动，点击next

注意：作为初学者，将此处Create module-info.java file 的对勾取消，这个我还不知道是为什么，反正因为它我写的代码一直不能运行，也在网上搜索了，但是都不是特别满意，还是请大佬们解答
点击Finish，新建的项目就建好啦！
接下来右击src→New→Class，就会出现下图

Name填写类名，注意：此处填写的名字必须要和代码内的class后面的名字一模一样
点击finish，就可以在编辑栏内敲下代码啦，我的第一个代码如下：
public class HelloChina {
public static void main(String[] args) {
System.out.println(“Hello,China!”);
}
}
点击上方工具栏里那个绿色小箭头run，此次代码就在控制台输出出来啦！如果还想在控制台输出其他的，可以将引号里的内容随便替换。

最后备注：
1.Java是严格区分大小写的语言，在没有经验敲下第一个代码时，尤其注意关键字（String，System）等的大小写，否则程序是不能正常运行的。
2.在敲代码时，一定要注意class后面的类名一定要与建class文件时的名字保持一致，否则代码是不能运行的。
3.和C语言一样，每一句结束也要有“;” 不能丢!
